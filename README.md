# vue-jsonschema-form

参考[react-jsonschema-form](https://github.com/rjsf-team/react-jsonschema-form)，基于 Vue3 开发的能够使用 **JSON Schema** 方式构建和自定义 Web 表单的简易组件。

## 实现功能

- 自定义表单校验
- 自定义表单组件开发
- 自定义表单样式
- 表单主题配置
- 多语言支持

具体案例参考项目的中的 demo
